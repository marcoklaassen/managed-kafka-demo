# managed-kafka-demo Project

## Description

This is a sample quarkus app with a 
* Kafka Producer

and a

* Kafka Consumer.

This app is configured to use the [Managed Kafka Service from Red Hat](https://console.redhat.com/application-services/streams/kafkas).
